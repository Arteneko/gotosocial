What is GoToSocial?
===================

GoToSocial will solve your problem of where to start with documentation,
by providing a basic explanation of how to do it easily.

Look how easy it is to use::

    import project
    # Get your stuff done
    project.do_stuff()

Features
--------

- Be awesome
- Make things faster

Installation
------------

Install $project by running:

    install project
